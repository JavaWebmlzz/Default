## 团队成员组成与分工

本项目由四位成员共同完成，各成员分工如下：

### 👤 成员 A —— 广告管理平台负责人

* 负责广告管理网站整体架构设计
* 实现后台管理员登录、权限认证模块
* 完成广告增删改查（CRUD）接口及后台 UI
* 负责广告发布 API（JS SDK / iframe）的设计与实现
* 负责广告投放规则引擎（分类、设备、频次等）

---

### 👤 成员 B —— 购物网站负责人

* 负责购物网站整体页面结构与分类体系
* 在购物网站中集成广告管理平台提供的广告嵌入脚本
* 配合测试“精准投放”—根据商品类型加载对应广告
* 完成匿名用户访问下的广告展示与 cookie/session 支持

---

### 👤 成员 C —— 视频分享网站负责人

* 负责视频分享网站实现
* 设计并实现视频广告插播机制：前贴片 / 中插 / 后贴片
* 支持定时插播逻辑（例如每 10 分钟播放一次广告）
* 基于 MIME 或分段播放（读主体视频 → 读广告视频）实现前端播放逻辑
* 与广告管理后台协同完成视频广告数据格式与播放配置

---

### 👤 成员 D —— 新闻网站负责人

* 搭建新闻网站页面框架与分类体系
* 负责在新闻网站中嵌入广告 API，并支持上下文分类投放
* 负责测试匿名用户场景下的广告展示
* 协助实现跨域 tracking 策略（如 iframe + postMessage）

---
